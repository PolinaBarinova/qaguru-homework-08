VKVideo_test_01: Воспроизведение видео, открытого по ссылке неавторизованным пользователем
Предусловие: Открыть браузер со страницей https://vkvideo.ru
1. Перейти по прямой ссылке на видео (по прямой ссылке на видео с таймкодом)
ОР: Видео по ссылке доступно и воспроизводится

VKVideo_test_02: Проверка наличия в результах поиска видео соответсвующего поисковому запроса
Предусловие: Открыть браузер со страницей https://vkvideo.ru
1. Ввести в поисковую строку значение 'кости сезон 1'
2. Нажать клавишу Enter
ОР: В результатах поиска доступны видео релевантные поисковому запросу
Постусловие: Очистить поискову строку

VKVideo_test_03: Проверка воспроизведения видео, открытого из плейлиста неавторизованным пользователем
Предусловие: Открыть браузер со страницей https://vkvideo.ru
1. Открыть раздел видео (Фильмы, Шоу, Детям)
2. Перейти к плейлисту из выбранного раздела
3. Открыть видео из плейлиста
ОР: Открыто выбранное видео, видео воспроизводится
